### 1、基础类型 

```javascript
fn dfs(path) {
  // 1、出口
  if ... return
  // 2、循环递归
  for {
    // 3、判断是否在路径里
    if ... continue
    // 4、加入路径、向下递归
    path.push
    dfs(path)
    // 5、完成后从路径弹出
    path.pop
  }
}
```
